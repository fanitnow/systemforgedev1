# Brand Guidelines

## Identity

**Name:** SystemForge AI  
**Tagline:** Built in 48 hours. Running forever.  
**Logo Concept:** Forge Mark — architectural rooftop/anvil shape inside a square border  
**Domain:** systemforgeai.com

---

## Color System

| Name | Hex | Usage |
|------|-----|-------|
| Forge Black | `#0A0A0A` | Primary background |
| Forge Gold | `#C9A84C` | CTAs, accents, logo, highlights |
| Gold Light | `#E8C96A` | Hover states |
| Off White | `#F5F0E8` | Body text |
| Card Dark | `#161616` | Card backgrounds |
| System Green | `#4CAF78` | Success states, live indicators |

**Color rules:**
- Black is the canvas — everything lives on it
- Gold is reserved for action items and key numbers only
- Never use more than 2 colors in a single UI element
- Green only for success states, never for decoration

---

## Brand Rules — What Never to Do

- Never stretch or distort the logo
- Never use gold on gold or black on black
- Never substitute a different font in official materials
- Never add a tagline directly to the logo mark
- Never use purple, blue, or red as primary colors
- Never post low-quality or off-brand content on business channels
